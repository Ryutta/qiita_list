# Qiita Stock Searcher

Qiitaの「ストック」した記事をリスト化し、検索できるツールです。
ユーザー情報（アクセストークン）は`.env`ファイルで管理し、Gitにはコミットしないようにしています。

**注意:** Qiita API v2では、ユーザーが「いいね」した記事の一覧を取得する機能が提供されていないため、本ツールでは「ストック」のみを対象としています。

## 準備

### 1. 依存ライブラリのインストール

```bash
pip install -r requirements.txt
```

### 2. Qiitaアクセストークンの取得

1.  Qiitaにログインします。
2.  [設定 > アプリケーション](https://qiita.com/settings/applications) にアクセスします。
3.  「個人用アクセストークン」の「新しくトークンを発行する」をクリックします。
4.  スコープは `read_qiita` にチェックを入れてください。
5.  発行されたアクセストークンをコピーします。

### 3. 環境変数の設定

プロジェクトのルートディレクトリに `.env` ファイルを作成し、以下のように記述してください。

```bash
QIITA_ACCESS_TOKEN=あなたのアクセストークン
```

※ `.env` ファイルは `.gitignore` に含まれているため、Gitにはコミットされません。

## 使い方

### 基本的な使い方

アクセストークンを設定していれば、引数なしで自分の記事を取得できます。

```bash
python -m src.main
```

または、スクリプトを直接実行する場合：

```bash
python src/main.py
```

### 特定のユーザーの記事を取得

ユーザーIDを指定して実行することも可能です（公開されているストックのみ取得可能）。

```bash
python -m src.main <user_id>
```

### 検索モード

起動後、対話モードで検索キーワードを入力できます。
終了するには `q` を入力してください。

また、コマンドライン引数で検索キーワードを指定することもできます。

```bash
python -m src.main --search python
```
